donrails 1.1へようこそ。 donrailsは日記や、blogの作成をするためのrubyプログラムです。 記事の作成には、webからの入力のほか、AtomAPIを使用することができます。

特徴

  • 徹底したcache。htmlファイル、xmlファイルを生成することでrubyプロセスの増殖を避け、高速化しています。
  • AtomAPIに対応(WSSE認証に対応しています) -> donrails:AtomAPI
  • 検索機能。また検索結果をRDFとして出力
  • 複数の入力形式(hnf, rd, plain text, wiki, html)に対応
  • コメントおよびトラックバック機能。およびそれらへのSPAMへの対応
  • 柔軟なカスタマイズが可能
    • レイアウトを作成(nostes.rhtmlと、その中で指定するcss)することにより、見た目を 自由に記述可能です。
  • 管理画面での記述やコメントの変更

  • rails 1.1 and later.
  • sqlite or SQL server
    • sqlite, sqlite3, postgresqlで使うテーブルを作成するためのsql fileが本アーカイ ブに含まれています。

sqliteを使う場合のインストール例

  • railsをインストールする。
  • gemsをインストールする。
  • sqliteのライブラリがないときlibsqlite0-devをいれる $ sudo gem install sqlite-ruby
  • make $ make すると、railsのファイルと必要なツリーが構築される
  • security_extensions pluginのインストール 追加方法は、script/pluginコマンドを使用します。 rails directory以下に移動し、 
     ./script/plugin source http://svn.aviditybytes.com/rails/plugins/
 ./script/plugin install security_extensions
    を行います。
  • donrails/tool/donrails-setup.rb を実行する。 設定ファイルが作成される。 問題なければ、これで終了。
  • 設定ファイルの編集
    • rails/config/database.yml
    • rails/config/donrails_env.rb spam関連の変数をdonrails_env.rbに追加してください。 
      IP_RBL = [ 'niku.2ch.net', 'opm.blitzed.us', 'bsb.empty.us' ]
HOST_RBL = [ 'rbl.bulkfeeds.jp', 'sc.surbl.org', 'bsb.empty.us' ]
URL_LIMIT = 5
      のようにします。記述しない場合はデフォルト値が使われます。 antispam.rbのデフォルト値はなるべくそのとき知られたRBLが はいっていますが、保証いたしかねます。
  • ユーザが自分のやりたいようにいじるファイル(必須)
    • rails/app/views/notes/index.rhtml ("donrails/example/index.rhtml"にサンプルがあります)
    • rails/app/views/layouts/notes.rhtml ("donrails/example/notes.rhtml"にサンプルがあります)
    • rails/public/ 以下はスタティックなファイルを置くために自由においてかまいません。 directoryだけでアクセスすると、rails/public/index.html が参照されるので 、必要ない場合は消してください。railsの動作確認用のfileです。
    • 画像のuploadを行うには、 rails/public/images/dump のディレクトリを動作ユーザから書きこめるようにしておく必要があります。
    • アップロードされたファイルは、GET /nosts/show_image/1 などとすると、/images/dump/image-files.png などにリダイレクトされます。 リダイレクト先は $IMAGE_BASE_PATH で指定できます。これが /test/ のときは、/test/images/dump/image-files.png などにリダイレクトされます。
